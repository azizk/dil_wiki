A helper function that parses the special strings in g_membersTable.
Basic syntax: 
Member := Array | Array2 | OptionalNode | Node | Code
Array := Identifier "[]"
Array2 := Identifier "[?]"
OptionalNode := Identifier "?"
Node := Identifier
Code := "%" AnyChar*
Identifier

Params:

membersthe member strings to be parsed.

Returns:

an array of tuples (Name, Type) where Name is the exact name of the member and Type may be one of these values: "[]", "[?]", "?", "" or "%".
