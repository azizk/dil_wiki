  • class UriView ; [abstract]
    • Implements an RFC 2396 compliant URI specification. See this page for more information.

    The implementation fails the spec on two counts: it doesn't insist on a scheme being present in the UriView, and it doesn't implement the "Relative References" support noted in section 5.2. Note that IRI support can be implied by assuming each of userinfo, path, query, and fragment are UTF-8 encoded (see this page for further details).

    Use a Uri instead where you need to alter specific uri attributes.
