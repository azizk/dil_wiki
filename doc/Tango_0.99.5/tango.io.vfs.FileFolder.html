  • class FileFolder : VfsFolder;
    • Represents a physical folder in a file system. Use one of these to address specific paths (sub-trees) within the file system.

  • this (char[] path, bool create = false);
    • Create a file folder with the given name and path. The name itself should not include '.' or '/' characters, though the path can point at whatever it pleases.

    Option 'create' will create the folder when set true, and open an existing folder otherwise
