Synchronized, formatted console output. This can be used in lieu of true logging where appropriate.

Trace exposes this style of usage: 

        Trace.format ("abc {}", 1);             => abc 1
        Trace.format ("abc {}:{}", 1, 2);       => abc 1:2
        Trace.format ("abc {1}:{0}", 1, 2);     => abc 2:1


Special character sequences, such as "\n", are written directly to the output without any translation (though an output-filter could be inserted to perform translation as required). Platform-specific newlines are generated instead via the formatln() method, which also flushes the output when configured to do so: 

        Trace.formatln ("hello {}", "world");


Explicitly flushing the output is achieved via a flush() method 

        Trace.format ("hello {}", "world").flush;
