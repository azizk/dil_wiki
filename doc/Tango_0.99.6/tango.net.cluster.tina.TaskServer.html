  • class TaskServer : ClusterServer;
    • Extends the ClusterServer to glue cluster-rpc support together

  • this (InternetAddress bind, Logger logger);
    • Construct this server with the requisite attributes. The 'bind' address is the local address we'll be listening on

  • void start (bool reuse = false); [override]
    • Start the server

  • void service (IConduit conduit); [override]
    • Factory method for servicing a request. We just create a new TaskThread to handle requests from the client. The thread does not exit until the socket connection is broken by the client, or some other exception occurs.
